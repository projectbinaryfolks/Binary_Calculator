Como funciona a calculadora binária?

A calculadora soma dois números binários de 2 bits: 

A = A1 A0
B = B1 B0

Resultado = S2 S1 S0

Cada entrada pode representar um número de 0 a 3:

Binário Decimal
00 0
01 1
10 2
11 3

Exemplo

Para obter resultado 3, uma possibilidade é fazer:

A = 01₂ = 1
B = 10₂ = 2

S2 S1 S0 = 0 1 1
